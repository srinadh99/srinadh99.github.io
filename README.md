# Personal Website (Minimal Mistakes)

This repository contains a personal academic website built with the **Minimal Mistakes** Jekyll theme.

## Deploy on GitHub Pages

You can publish either from:

- **Root** (`/(root)`) — use the root-level `_config.yml`
- **docs** (`/docs`) — use the `docs/_config.yml`

In GitHub:

1. Repo → **Settings** → **Pages**
2. Choose **Deploy from a branch**
3. Select branch (`main` / `master`) and folder (`/(root)` or `/docs`)

## Customize

- Update links in `_config.yml` (and `docs/_config.yml` if using `/docs`)
- Edit pages in `_pages/` (and `docs/_pages/` if using `/docs`)
- Replace `assets/images/bio-photo.png` with your photo
